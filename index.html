Html tags

Tag Name Description
<h1>-<h6> Section Heading element Represents a paragraph. Represented as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation
<p> Paragraph element Tag needed for opening paragraphs
<div> Division element The generic container for flow content
<section> Section element a generic standalone section of a document, which doesn't have a more specific semantic element to represent it
<header> Header Represents introductory content, typically a group of introductory or navigational aids
<main> Main Represents the dominant content of the <body> of a document
<footer> Footer Typically contains information about the author of the section, copyright data or links to related documents
<span> The Content Span element A generic inline container for phrasing content

Html table tags

Tag Name Description
<table> The Table element Represents tabular data
<thead> The Table Head element Defines a set of rows defining the head of the columns of the table.
<tbody> The Table Body element encapsulates a set of table rows, indicating that they comprise the body of the table
<tfoot> The Table Foot element Defines a set of rows summarizing the columns of the table
<tr> The Table Row element Defines a row of cells in a table
<th> The Table Header element Defines a cell as header of a group of table cells
<td> The Table Data Cell element Defines a cell of a table that contains data

Html table attributes

Attribute Name Description
colspan Colum span Indicates for how many columns the cell extends
rowspan Row span Indicates for how many rows the cell extends
